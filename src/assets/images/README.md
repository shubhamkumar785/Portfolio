# Image Assets

Please add your images to this directory with the following names:

## Required Images:

1. **profile.jpg** or **profile.png**
   - Your profile/avatar image
   - Recommended size: 430px × 480px
   - Will be displayed with rounded corners

2. **hero.jpg** or **hero.png**
   - Main center hero image (person looking at landscape)
   - Recommended size: 680px × 850px
   - Should be a high-quality image

3. **travel1.jpg** or **travel1.png**
   - First travel image (appears on left/bottom)
   - Recommended size: 320px × 420px

4. **travel2.jpg** or **travel2.png**
   - Second travel image (appears on right/top)
   - Recommended size: 290px × 460px

5. **logo1.png** and **logo2.png**
   - Company logos (grayscale recommended)
   - Recommended height: 60px
   - Transparent background preferred

Once you add these images, update the component imports.
